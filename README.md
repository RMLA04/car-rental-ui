#Car-rental-ui
# 🚗 Car Rental Management System

A full-stack database project built using **PostgreSQL**, **Flask**, and **HTML/CSS**, designed to manage car rentals, customers, payments, and reports.

---

🔗 [https://car-rental-ui-7.onrender.com/]

---

## 📂 GitHub Repository

🔗 https://github.com/RMLA04/car-rental-ui

---

## 🧾 Project Description

The Car Rental Management System is a web-based application that allows users to manage:

- Customers
- Cars and categories
- Rental transactions
- Payments
- Reports and analytics

The system connects to an online PostgreSQL database and provides a user-friendly interface for performing all operations.

---

## ⚙️ Technologies Used

- **Backend:** Python (Flask)
- **Database:** PostgreSQL (Supabase)
- **Frontend:** HTML, CSS (Jinja2 templates)
- **Deployment:** Render
- **Version Control:** Git & GitHub

---

## 🗄️ Database Features

- Relational database design
- Primary & foreign keys
- Constraints (CHECK, UNIQUE)
- Stored function (`calculate_rental_amount`)
- Triggers (auto-update car status)
- Indexing for performance

---

## 💻 System Features

- Add and manage customers
- Add and view cars
- Create and complete rentals
- Automatic rental cost calculation
- Record payments
- View reports (revenue, most rented cars)

---

## 🔐 Security

- Database credentials managed via environment variables
- No sensitive data stored in the repository

---

## 🚀 Deployment

- **Frontend + Backend:** Render
- **Database:** Supabase PostgreSQL

The system is accessible online via a public URL.
